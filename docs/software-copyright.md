开发目的是为求职者提供AI驱动的简历优化与模拟面试训练，通过智能技术提升求职成功率。
面向领域为求职招聘领域，适用于简历优化、AI模拟面试及职业技能提升场景。
软件主要功能如下：本软件是一款集简历优化、模拟面试、专项训练、面试复盘于一体的智能求职辅助平台。简历管理方面，支持用户在线填写简历内容，同时提供PDF格式简历上传功能，上传后系统自动保存简历文件，并支持在页面中嵌入式预览PDF原件，预览清晰且不可编辑，方便用户随时查看和管理自己的简历信息。简历优化方面，系统基于AI技术对用户简历进行智能分析，从面试官视角给出综合评分与逐条优化建议，用户一键启动AI分析后系统自动输出百分制评分和详细的修改建议，涵盖简历结构、项目描述、技能表述等多个维度，帮助用户精准提升简历质量。模拟面试方面，提供两种面试模式，题库模式从系统知识库中抽取原题进行针对性练习，考察基础知识的掌握程度，智能模式基于知识库内容由AI进行改写和变式训练，从不同角度考察同一知识点，提升应变能力，用户可自由选择题目数量，面试过程中AI逐题评分、给出详细评价和标准参考答案，并支持针对用户回答进行深度追问，层层深入考察，真实还原面试场景，面试完成后系统自动生成综合评价，涵盖整体表现分析。专项训练方面，基于用户上传的简历内容进行智能化出题，系统通过PDF文本提取技术自动获取简历中的技术栈、项目经验、专业技能等信息，由AI从资深面试官角度生成针对性的面试题目，出题策略严格确保问题紧扣简历中出现的具体技术内容，考察技术深度和实践能力，避免重复出题，同样支持题数选择、AI逐题评分、评价反馈与追问功能，帮助用户针对自身简历查漏补缺。面试复盘方面，每次面试结束后AI自动对全部问答记录进行综合分析，生成面试总结，包括总体评分、表现优点、不足之处、改进建议和知识盲区分析。面试历史方面，记录用户所有的模拟面试和专项训练记录，支持按时间倒序查看。错题本方面，自动收录用户在面试中答错的题目，支持按分类筛选和查看，用户可对错题进行复习，系统记录复习次数和上次复习时间。知识库管理方面，管理员可通过Excel批量导入面试题目，支持题目、答案、分类等字段的导入，系统提供题库管理和分类管理功能，管理员可对题目进行增删改查并查看导入日志。数据统计方面，通过可视化图表展示用户的学习数据，包括模拟面试次数、总题数、复盘数和错题数等关键指标。用户系统方面，支持注册登录功能，采用JWT令牌认证和密码加密存储保障数据安全，区分普通用户和管理员角色，管理员享有题库管理、分类管理等高级权限。AI对话引擎方面，系统集成DeepSeek和Ollama两种AI模型并支持灵活切换，所有简历优化、题目生成、回答评分、面试总结等AI功能均通过该引擎驱动，确保智能化服务质量。
